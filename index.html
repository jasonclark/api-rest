Welcome to an example REST API for teaching and learning.

This sample API uses A Tribe Called Quest lyrics. Code is available on GitHub.

"List all resources"
GET => retrieve and list all resources
URI: /api/resources
"List single resource"
GET => find and retrieve resource by id
URI: /api/resource/id
"List single random resource"
GET => retrieve random resource
URI: /api/resource/random
"Search resources"
GET => search and retrieve resources by matching string in text field
URI: /api/resources/search?q=ADD-QUERY-HERE
